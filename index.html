What I've done

Pune Corona Tracker

See on The Play Store
BEHIND THE SCENES
WHAT THIS IS:

An Android App made in Kotlin to track the statistics of Corona spread in the City of Pune,Maharashtra

GOAL:

Make a visually appealing Corona Tracker, something that's different from all other already present COVID Trackers

LEARNING POINTS:

Make an Android App that shows all possible info about COVID-19 in my home location. Additional Perks: Easy to Use, Lag Free and as conducive as possible

  • Search for an API that sends out data (region specific).If none, then create one
    • Learned to sniff websites and found out a non conventional data source
    • Many sections each handling a different aspect about the disease
    • Add a map, showing red zones
  • One of the most interesting challenges was that I had to design most of the graphics by myself,deal with the hassle of changing data sources.I also learnt about doing background tasks like downloading data, updating it without disturbing the user so as to not spoil the user experience The most interesting part is that this project started as a fun idea by Nikhil Kharade but resulted in a grand success with 158 users and still counting
